# Pod-Template
可以快速创建支持源码和 Framework 的 Pod库模板

```shell
git clone https://github.com/InsectQY/Pod-Template.git && cd Pod-Template && $PWD/GBPods-Template-CommandLine && open ./
```
## 需要的安装工具
- Carthage
- Cocoapods
